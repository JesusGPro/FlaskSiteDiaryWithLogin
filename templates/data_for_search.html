Introduce the SQL command to look for records

1) To look for a single word in any column introduce:

SELECT * FROM USER_JGP WHERE CONTENT LIKE '%gantry%' ORDER BY ID

2) Looking in one field only, you have theree: ID, TITLE, CONTENT and DATE_TIME:

SELECT CONTENT FROM USER_JGP WHERE CONTENT LIKE '%gantry%' ORDER BY ID

3) Looking diferent words in one column, the column is CONTENT, you can pick a different one:

SELECT * FROM USER_JGP WHERE CONTENT like '%C29%' and CONTENT like '%gantry%' and CONTENT like '%completed%'

4) If you want to look for a date, introduce the date with "_" instead of "/", in the url will not work:

SELECT * FROM USER_JGP WHERE TITLE like '%2024_02_03%' ORDER BY ID
